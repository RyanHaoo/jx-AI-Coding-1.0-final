# Quickstart - Agent 对话模块（Mock 实现）

## 前置条件

- 已配置 Supabase 与 OpenRouter 相关环境变量
- 已有可登录用户，且用户具备至少一个项目身份

## 启动

```bash
npm run dev
```

## 验证步骤

1. 登录并进入 `/mobile/assistant`，确认首屏出现欢迎语。  
2. 发送任意文本，确认出现加载态并流式收到回复。  
3. 输入“查一下我的工单”，确认返回结构化 mock 工单数据。  
4. 输入知识问题（如“防水施工规范”），确认返回 Markdown 格式答案。  
5. 输入创建工单描述（如“3号楼5层卫生间漏水”），确认出现 create_ticket 卡片，填写责任人后提交，确认卡片切换只读并提示创建成功。  
6. 切换为施工方或管理员身份，重复创建工单指令，确认被拒绝且不出现卡片。  
7. 连续发起超过 6 轮问答，确认最早轮次被裁剪。  
8. 刷新页面，确认按当前登录用户 ID 恢复最近 6 轮历史。

## 质量闸门

```bash
npm run lint
npx tsc --noEmit
```

# PC 端 — 通用布局

**适用角色：** 管理员

---

## 1. 整体结构

```
┌──────────────────────────────────────────────┐
│                   顶栏 (Top Bar)              │
├──────────────┬───────────────────────────────┤
│              │                               │
│  左侧菜单     │        页面内容区域             │
│  (Side Nav)  │     (各页面自定义内容)           │
│              │                               │
└──────────────┴───────────────────────────────┘
```

---

## 2. 顶栏（Top Bar）

**位置：** 固定在页面顶部，全宽

**内容：**
- 左侧：应用标题 / Logo（「建筑施工质检情报员」）
- 右侧：用户组件（用户头像 + 姓名，点击后下拉菜单含退出登录选项，见[核心组件/用户组件.md](../核心组件/用户组件.md)）

---

## 3. 左侧菜单（Side Navigation）

**位置：** 固定在页面左侧，竖向导航

**菜单项：**

| 图标 | 菜单名 | 路由 |
|------|--------|------|
| 📊 | 数据大盘 | `/dashboard/overview` |
| 📋 | 工单中心 | `/dashboard/tickets` |
| 📚 | 知识运营 | `/dashboard/knowledge` |

**激活状态：** 当前页面对应菜单项高亮显示。

---

## 4. 页面内容区域

右侧主内容区，各页面自行定义内容。

---

## 5. 鉴权守卫

所有 PC 端页面（`/dashboard/*`）均需管理员登录。若非管理员角色访问，跳转到 `/login?redirect=/dashboard/xxx` 或展示无权限提示。

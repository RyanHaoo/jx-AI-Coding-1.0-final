# 移动端 — 通用布局

## 1. 整体结构

移动端页面由以下固定区域组成：

```
┌─────────────────────────┐
│         顶栏 (Top Bar)   │
├─────────────────────────┤
│                         │
│      页面内容区域        │
│    (各页面自定义内容)    │
│                         │
└─────────────────────────┘
```

**无底栏导航。** 移动端不使用底部 Tab Bar，两个核心功能作为独立页面，通过侧边目录切换。

---

## 2. 顶栏（Top Bar）

**位置：** 固定在页面顶部

**内容：**
- 左侧：展开按钮（汉堡菜单图标），点击后打开侧边目录
- 中间：当前页面标题
- 右侧：预留（暂无内容）

---

## 3. 侧边目录（Side Drawer）

点击顶栏左侧展开按钮后，从左侧滑入。

**内容结构：**

```
┌──────────────────────┐
│  🫧 建筑施工质检情报员  │  (应用标题)
├──────────────────────┤
│                      │
│  💬 智能助手          │  → /mobile/assistant
│  📋 工单列表          │  → /mobile/tickets
│                      │
├──────────────────────┤
│  [用户组件]           │  (当前用户信息，见用户组件.md)
│  退出登录 / 切换身份   │
└──────────────────────┘
```

**交互：** 点击任意菜单项后，侧边目录自动关闭并跳转到对应页面；点击蒙层也可关闭。

---

## 4. 页面过渡

- 进入工单详情页时，采用向左滑入动画
- 返回列表页时，采用向右滑出动画
- 打开侧边目录时，从左侧滑入；关闭时向左滑出

---

## 5. 鉴权守卫

所有移动端页面（`/mobile/*`）均需登录状态。未登录时自动跳转到 `/login?redirect=/mobile/xxx`，登录完成后跳回原页面。

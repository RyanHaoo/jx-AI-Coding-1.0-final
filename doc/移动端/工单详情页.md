# 移动端 — 工单详情页

**路由：** `/mobile/tickets/:id`

---

## 1. 页面概述

展示单个工单的完整信息，并根据当前用户身份展示操作按钮。

---

## 2. 页面结构

```
┌─────────────────────────┐
│   顶栏（← 返回 | 工单详情） │
├─────────────────────────┤
│                         │
│     工单详情组件          │
│  （见核心组件/工单组件.md） │
│                         │
└─────────────────────────┘
```

**顶栏：** 左侧有返回按钮，点击后返回工单列表页（`/mobile/tickets`）。

---

## 3. 核心内容

页面内容完全由**工单详情组件**承载，详细字段定义、编辑/展示状态切换、操作按钮逻辑见：

→ [核心组件/工单组件.md](../核心组件/工单组件.md)

---

## 4. 数据加载

- 进入页面时根据路由参数 `id` 请求工单详情 API
- 加载中显示骨架屏
- 若工单不存在或无权访问（不属于当前项目），展示 404 提示并提供返回按钮
